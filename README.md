# 🍕 Pizza Sales Analysis | SQL + Excel

## Project Overview
Analysis of transactional pizza sales data to evaluate revenue performance, customer ordering patterns, and product contribution.

## Tools Used
- **SQL (MySQL)** — Data cleaning, transformation, and business analysis
- **Excel** — KPI dashboard and visualization

## Key Metrics
| Metric | Value |
|--------|-------|
| Total Revenue | $817K+ |
| Total Orders | 21K+ |
| Total Pizzas Sold | 49K+ |
| Average Order Value | $38.31 |
| Average Pizzas per Order | 2.32 |

## Key Insights
- 🗓️ Orders peak on Fridays and weekends
- ⏰ Two peak windows — 12-1 PM and 5-8 PM
- 🍕 Large size dominates at ~46% of total sales
- 🏆 Classic category drives highest revenue
- 📉 Worst sellers tested before removal
